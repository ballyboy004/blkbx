# BLACKBOX Product Spec V1

## Goal
Ship BLACKBOX Campaign V1.

Purpose:
Generate and execute release campaign infrastructure for independent artists.

## Core User Flow

1. Artist creates campaign
2. System generates strategy
3. System generates campaign assets
4. Artist reviews + approves
5. Campaign tasks are executed

## Campaign Assets Generated

- Release strategy
- Press release
- Email announcement
- First-week social captions
- Campaign task list

## Success Criteria

- Artist can create campaign
- Strategy generates successfully
- Assets generate correctly
- User can approve outputs
- Campaign progress is trackable
